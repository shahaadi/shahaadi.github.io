A few examples of recent projects

re-paragrapher

Browser extension that uses natural language processing principles to "reparagraph" onscreen text upon click. Helps make blocky text more readable.

In Development

Overhead Movement Alarm Radar

Prototype using ultrasonic sensors and vibration motors to help the visually impaired detect overhead obstacles and their direction via haptic feedback.

In Development

Cog*Ify

App using a CNN trained on NCS song spectrograms and ‘most replayed’ data from YouTube to predict the likely ‘most popular’ portion of each song in a Spotify playlist and order and mix them with cosine similarity and crossfade.

Check it out!

Personal Site

Well designed, clean and professional personal website with portfolio created using HTML, CSS, and JavaScript, and without any frameworks. Has options for both light and dark mode.

Check it out!

dig-interpret

Device using a convolutional neural network and text-to-speech to help the visually impaired audibly perceive numerical data in their environment. Overhauled from October to December 2021 to interpret all text using optical character recognition; more info to be published at a later date.

Check it out!
